Команда git grep ищет совпадение с указанной строкой во всех файлах проекта.
Это очень удобная команда для быстрого анализа из терминала. Она удобнее обычного
grep, так как знает про игнорирование и не смотрит в директорию .git,
а ещё умеет искать по истории:

> git grep a // для наглядности ищем букву a во всех файлах проекта

// Консоль

file1.txt:'lsadjf;jalkhsdfkja;dslkfn;asldknf;akjj;aksdf;kaasdas
file2.txt:asdas
