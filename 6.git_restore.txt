Для отката измений в файле можно использовать команду git restore <Имя файла>

Создадим новый файл в проекте

~ echo 'some data' > anotherFile.txt

Добавим его для отслеживания и закомитим

~ git add anotherFile.txt
~ git commit -m 'added another file'

сделаем в нем изменения

// сделали изменения в файле

и проверим статус

~ git status

упс.. оказалось что мы внесли нежелательные изменения и хотим его откатить
не проблема!

~ git restore ./anotherFile.txt

Готово теперь наш файл откатился то состояния передыщего коммита, а все изменения
сделанные в файле после предыдучшего помита были удалены
